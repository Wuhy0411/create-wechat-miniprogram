# create-wechat-miniprogram

> 微信小程序从零到发布上线的新手完整指南 — Reasonix Skill

一个专为 **零基础新手** 设计的 Reasonix Skill，从注册账号到发布上线，按步骤引导你创建属于自己的微信小程序。

---

## 适用人群

- 第一次创建微信小程序，不知道从哪里开始的**纯新手**
- 手上有小程序项目代码，想快速跑起来并发布的**开发者**
- 需要配置云开发（数据库 + 云函数）的**自学者**

---

## 安装方法

### 方式一：从 GitHub 安装（推荐）

在 Reasonix 中执行以下命令安装此 skill：

```
install_source --source https://github.com/Wuhy0411/create-wechat-miniprogram.git
```

### 方式二：手动复制

将 `SKILL.md` 复制到你的 Reasonix skills 目录：

```bash
# 全局安装（所有项目可用）
cp SKILL.md ~/.reasonix/skills/create-wechat-miniprogram/

# 或当前项目安装（仅当前项目可用）
cp SKILL.md .reasonix/skills/create-wechat-miniprogram/
```

---

## 使用方法

### 在 Reasonix 会话中

直接输入以下命令调用 skill：

```
/create-wechat-miniprogram
```

Reasonix 会加载完整指南，然后你可以告诉它你当前做到了哪一步，它会带你继续。

### 使用示例

| 你的问题 | Reasonix 会做什么 |
|---------|-----------------|
| "从第一步开始" | 逐步骤引导注册小程序账号 |
| "我已经有 AppID 了" | 跳到导入项目代码的步骤 |
| "云函数上传报错了" | 定位到 npm 权限问题解答 |
| "审核被拒了，怎么办" | 分析常见拒审原因给出解决方案 |
